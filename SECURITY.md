# Security Policy

This repository is a sanitized documentation showcase for ACRO. It is not a production system, deployable security product, or live service.

## Reporting Guidance

Please do not report issues as production vulnerabilities in this repository. There is no running ACRO service, customer environment, or live infrastructure represented here.

Appropriate reports include:

- accidentally committed secrets,
- real customer or incident data,
- private infrastructure details,
- unsafe offensive instructions,
- broken links or misleading claims.

## Data Handling

Do not submit secrets, credentials, API keys, customer data, employee data, real incident data, malware samples, exploit payloads, or private logs in issues, pull requests, screenshots, or examples.

All examples in this repository should remain synthetic, sanitized, and documentation-only.
